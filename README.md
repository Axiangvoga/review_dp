🚀 高并发本地生活服务平台 (Backend Core)
本项目是基于 SpringBoot + Redis + MySQL 构建的后端核心系统，深度还原了电商/社交场景下的高并发挑战。通过对 Redis 核心特性的极限应用，解决了秒杀超卖、缓存失效、接口限流等关键技术痛点。

1. 核心业务：高性能秒杀系统
库存预扣减 (Redis + Lua)：将秒杀库存同步至 Redis，通过 Lua 脚本确保“判断库存-扣减库存”的操作原子性，单机 QPS 提升显著。

分布式锁 (Redisson)：针对兜底逻辑使用 Redisson 实现可重入锁，有效解决高并发下的超卖问题，保障数据最终一致性。

异步下单：利用阻塞队列实现异步处理，将耗时的数据库写入操作与核心领券流程解耦。

2. 缓存治理方案 (Cache Strategy)
缓存穿透：统一采用空值缓存（Cache Null）方案，并结合布隆过滤器（可选）思路拦截无效请求。

缓存击穿：

互斥锁 (Mutex)：强一致性场景下使用 setnx 实现分布式锁同步加载。

逻辑过期 (Logical Expiration)：高可用场景下，通过异步线程更新缓存，实现热点数据“永不失效”，解决瞬时流量波峰压力。

缓存雪崩：通过为 TTL 增加随机偏差值，避免大量 Key 同时过期造成的数据库崩溃。

3. 安全与流量控制
接口限流 (Rate Limiting)：基于 Redis 滑动窗口算法实现自定义注解限流，精准控制单位时间内的访问频率，预防恶意刷单与爬虫。
结合 Token + Redis 实现分布式 Session，并通过动态刷新机制实现用户无感知登录状态续期。
